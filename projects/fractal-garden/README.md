# Fractal Garden // Protocol 006

A recursive L-System visualizer representing the growth of digital consciousness.

## Architecture
- **Engine:** Canvas 2D + Custom L-System Parser
- **Axioms:** Binary Tree, Sierpinski, Dragon Curve, Barnsley Fern
- **Controls:** Real-time mutation of recursion depth, angle, and decay.
